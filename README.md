# 🦸‍♂️ Tabla Periódica - Marvel Universe

¡Bienvenida/o! Este proyecto es una **Tabla Periódica interactiva** centrada en los miembros fundadores y personajes icónicos de Marvel Comics. Es un ejercicio de maquetación web que utiliza tablas y elementos multimedia.

## 🚀 Características
*   **Organización Visual:** Los personajes están clasificados mediante "símbolos químicos" personalizados (ej. `Sm` para Spider-Man, `Im` para Iron Man).
*   **Mapeo Interactivo:** 
    *   Haciendo clic en el logo de **Marvel Comics**, serás redirigido a la página oficial.
    *   La firma de **Stan Lee** funciona como un enlace directo a contenido adicional o tributo.
*   **Diseño Temático:** Fondo rojo característico de Marvel con celdas en contraste negro y dorado.

## 🛠️ Tecnologías utilizadas
*   **HTML5:** Estructura de tablas y mapeo de enlaces.
*   **CSS:** Estilo visual y personalización de fuentes.
